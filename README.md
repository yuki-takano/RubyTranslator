# Ruby Translator

## 必要環境

- iOS Version 13.2~
- Xcode 11.3.1
- Swift5.0
- CocoaPods 1.8.4

## アプリの機能
- 入力した文字列をひらがなに変換して表示します。  

## 利用イメージ
![demo](https://user-images.githubusercontent.com/6592495/77228257-62e51400-6bc9-11ea-9bc4-eb52053acb90.gif)

## セットアップ

1. `cd {path to this repository}`
2. `pod install`
